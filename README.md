# Crisis Communication & PR Response Generator

**Your brand's reputation repair kit — holding statement to media prep in minutes.**

When a crisis hits, you have 30-60 minutes to respond before the story gets written without you. This skill generates professional crisis communication across every channel: holding statement, public statement, internal employee communication, and media Q&A prep — for any business crisis scenario.

## What You Get

| Prompt | Output | Use When |
|--------|--------|----------|
| 1. Holding Statement | 3-5 sentences, 3 tone variants, do-not-say list | First 30-60 min of any crisis |
| 2. Public Statement | 300-500 word statement + leadership quote + FAQ | Hour 2-4: press, website, social |
| 3. Internal Communication | Employee notification + post-resolution update + social media guidance | Any time staff need to know |
| 4. Media Q&A Prep Kit | 10 hardest journalist questions + approved answers + sound bites | Before any press interview |

1. Copy the prompt for the output you need
